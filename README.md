# 한국어 ↔ 외국어 발음·뜻 변환기

외국어(영어 · 비사야어(세부아노) · 따갈로그어)를 입력하면 **한국 발음 / 한국어 뜻**으로,
한국어를 입력하면 **로마자 발음 / 외국어 뜻**으로 변환해 주는 웹앱입니다.
단어뿐 아니라 문장도 변환할 수 있습니다.

## 변환 예시

| 언어 선택 | 입력 | 결과 |
|---|---|---|
| 영어 | `tree` | `트리/나무` |
| 영어 | `나무` | `namu/tree` |
| 비사야어 | `Salamat po` | `살라맛 뽀/감사합니다` |
| 비사야어 | `감사합니다` | `gamsahapnida/salamat` |
| 따갈로그어 | `Magandang umaga po` | `마간당 우마가 뽀/좋은 아침이에요` |

## 사용 방법

정적 웹페이지라 별도 설치가 필요 없습니다.

```bash
# 저장소를 받은 뒤 아무 정적 서버로 열면 됩니다
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

`index.html`을 브라우저에서 바로 열어도 동작하며, GitHub Pages로 배포해도 됩니다
(저장소 Settings → Pages → 브랜치 선택).

1. 언어(영어/비사야어/따갈로그어)를 선택합니다.
2. 단어나 문장을 입력합니다. 한글이 들어 있으면 자동으로 한국어 → 외국어 방향으로 변환합니다.
3. **변환하기** 버튼(또는 Ctrl/⌘+Enter)을 누르면 `발음/뜻` 형태로 결과가 나옵니다.

## 동작 원리

- **발음(음역)** — 브라우저 안에서 규칙 기반으로 즉시 변환합니다 (`translit.js`).
  - 외국어 → 한글: 영어는 철자-발음 규칙 + 자주 쓰는 단어 사전, 비사야어/따갈로그어는
    철자가 발음과 거의 일치하므로 음운 규칙으로 변환 (무성 파열음 p/t/k는 된소리 ㅃ/ㄸ/ㄲ 표기)
  - 한국어 → 로마자: 국어의 로마자 표기법 기반 음절 단위 변환
- **뜻(번역)** — 무료 번역 API(Google 비공식 엔드포인트, 실패 시 MyMemory)를 사용하므로
  인터넷 연결이 필요합니다. 번역이 실패해도 발음 변환 결과는 표시됩니다.
- **듣기(음성)** — 브라우저 내장 음성 합성(Web Speech API)으로 발음과 뜻을 읽어 줍니다.
  - 한글 발음·한국어 뜻은 한국어 음성으로, 영어 뜻은 영어 음성으로 재생
  - 한국어 → 외국어일 때 발음 듣기는 로마자 대신 원래 한국어를 읽어 실제 발음을 들려줍니다
  - 비사야어·따갈로그어 뜻은 필리핀어(fil-PH) 음성을 사용하며, 기기에 해당 음성이 없으면
    브라우저 기본 음성으로 재생됩니다

발음 표기는 규칙 기반 근사치라 실제 발음과 다를 수 있습니다.

## 파일 구성

| 파일 | 역할 |
|---|---|
| `index.html` | 페이지 구조 |
| `style.css` | 스타일 |
| `translit.js` | 음역/로마자 변환 로직 (Node에서도 사용 가능) |
| `app.js` | UI 동작 + 번역 API 연동 |
